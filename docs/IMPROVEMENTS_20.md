# 20 项改进清单（已实现）

## 手机适配（10 项）✅
1. **移动端技能快捷按钮** — 游戏中左下角 Q/E/R/X/N 可点击，与 HUD 冷却同步
2. **触摸移动死区与平滑** — 18px 死区 + 0.18 平滑插值，避免抖动
3. **竖屏 HUD 折叠** — 竖屏下 HUD 单行可横向滚动
4. **射击区单指响应** — firePointerId 仅首指生效
5. **横屏/全屏引导** — 开始界面移动端显示「横屏 · 全屏体验更佳」
6. **移动端音效开关** — 开始界面「音效」勾选 + localStorage 持久化
7. **触摸震动反馈** — 技能按钮按下时 navigator.vibrate(10)
8. **长按射击防误触** — 射击按钮按下 80ms 后才开启连射
9. **横屏安全区重排** — 横屏矮屏下技能行缩小、底部安全区
10. **性能模式** — 「低特效」勾选减少粒子数，Boss 击破粒子同步减少

## 游戏本体（10 项）✅
1. **关卡标题** — 开局与过关后 canvas 绘制 "STAGE N" 约 2 秒
2. **Boss 血条数字** — 血条下方显示 "当前/最大" HP
3. **暂停「返回标题」** — 暂停菜单「继续」「返回标题」双按钮
4. **设置持久化** — 难度/机体/静音/低特效 localStorage 记住并 applySettings 加载
5. **波次提示** — 与关卡标题合并为 STAGE N 展示
6. **道具拾取浮动字** — 拾取 P/M/N 显示 "+POWER" "+MISSILE" "+NUKE"
7. **NEW RECORD 动画** — CSS new-record-pulse 缩放动画
8. **敌弹描边** — 敌机子弹绘制后 stroke 白色半透明描边
9. **连击音效** — playCombo() 双音，连击≥5 时触发
10. **Boss 击破音效** — playBossKill() 与普通爆炸区分
